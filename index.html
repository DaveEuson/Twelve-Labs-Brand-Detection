Set a global default viewership as your baseline. Entries with more or fewer viewers will scale the calculated value proportionally. Leave at 0 to disable.

Example: default 500K viewers. An entry with 1M viewers gets ×2.0 multiplier. An entry with 100K gets ×0.2. Override per-entry by clicking the 👥 column in the log.
